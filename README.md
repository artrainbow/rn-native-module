# Requirements

- Node v20.5.0
- Yarn 1.22.19

# Installation

```
yarn global add npx
```

```
yarn global add detox-cli
brew tap wix/brew
brew install applesimutils
```

```
yarn global add eslint
```

```
yarn
```

```
npx pod-install
```

# Run the project

## iOS:

```
yarn ios
```

## Android

```
yarn android
```

# Run tests

```
yarn test:unit
```

```
yarn test:e2e:ios
```
